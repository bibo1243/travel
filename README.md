# 2026 企業員工自強活動行程決策系統

這是一個專為企業員工旅遊設計的互動式行程決策網頁。整合了**阿里山**與**台東**兩大行程方案，提供詳細的時間軸、景點介紹、即時天氣資訊以及預算比較功能，協助同仁做出最佳選擇。

## 🌟 特色功能

- **互動式行程切換**：一鍵切換「阿里山」與「台東」行程，即時預覽不同方案。
- **精美時間軸設計**：
  - 每日行程採用垂直時間軸卡片呈現。
  - 整合 Unsplash 與官方高品質景點圖片。
  - 每個景點標題皆可點擊連結至官方網站或相關介紹頁面。
- **詳細資訊整合**：包含時間、活動內容、餐飲/住宿/交通類別標籤以及注意事項。
- **預算比較**：即時顯示兩者費用差異。

## 🛠️ 技術架構

- **HTML5**: 語意化標籤結構。
- **CSS3**: 
  - 使用現代 Flexbox 與 Grid 佈局。
  - Glassmorphism (玻璃擬態) 設計風格。
  - 響應式設計 (RWD)，支援手機與桌面瀏覽。
- **JavaScript (ES6+)**: 
  - 動態 DOM 操作與渲染。
  - 行程資料結構化 (JSON-like objects)。
- **FontAwesome**: 介面圖示支援。

## 🚀 如何執行

1. 將專案 Clone 至本地端：
   ```bash
   git clone https://github.com/your-username/company-trip-2026.git
   ```
2. 直接使用瀏覽器打開 `index.html` 即可瀏覽。

## 📂 檔案結構

- `index.html`: 主頁面結構。
- `style.css`: 樣式表與動畫效果。
- `script.js`: 行程資料與互動邏輯。
- `*.png`: 行程預覽圖 (原始資料參考)。
